# Общие обозначения 
task - Задача
branch - Набор изменений
Слить изменения - Завершить правки
master - Основная версия документа
tag - Указатель

# Задачи
create_repos - Создать документ
create_branch - Начать правки
insert_files - Добавить страницы
remove_fies - Удалить страницы
set_text - Задать текст
reset_text - Убрать текст
merge_branch - Завершить правки

# Статусы задач
new - Создается
queued - Ожидает
running - В процессе
success - Готово
error - Ошибка
